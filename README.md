# Cálculo Per Capita

Projeto em Java para calcular a renda per capita familiar com entrada de dados pelo terminal.

## Objetivo

O programa recebe a renda familiar total e a quantidade de pessoas da família, calcula a renda por pessoa e exibe o resultado.

## Tecnologias

- Java
- Scanner
- Git e GitHub

## Como executar

Compile o arquivo principal:

```bash
javac src/Main.java
```

Execute o programa:

```bash
java -cp src Main
```

## O que foi praticado

- Entrada de dados pelo terminal
- Operações matemáticas simples
- Organização básica de um projeto Java
